## SpringHill

### Overview
SpringHill is a full-stack web application that combines the power of a Spring Boot backend with a Next.js frontend. The application utilizes PostgreSQL for database management and Apollo Client for GraphQL integration, creating a robust and modern web application framework.

### Features
- **Spring Boot Backend**: Provides a scalable and efficient server-side application with a RESTful API and GraphQL endpoints.
- **Next.js Frontend**: Delivers a fast, responsive, and SEO-friendly user interface with React and TypeScript.
- **PostgreSQL Database**: Manages data storage with reliable and powerful relational database capabilities.
- **GraphQL Integration**: Enables flexible and efficient data fetching with Apollo Client.
- **Dockerized Deployment**: Simplifies development and deployment processes using Docker and Docker Compose.
- **Tailwind CSS**: Ensures a beautiful and consistent design with utility-first CSS framework.
- **TypeScript Support**: Enhances code quality and developer experience with type safety and modern JavaScript features.

### Services
- **User Service**
- **Project Management**
- **Orders**
- **Notifications**
- **Meeting Scheduling**
- **Billing**

### Getting Started
1. **Clone the Repository**:
   ```sh
   git clone https://github.com/bunchoftrees/SpringHill.git
   cd SpringHill
2. **Set Up Environment**:
   - Docker and Docker Compose should be installed on your machine.

3. **Build and Start Application**:
   ```sh
   docker-compose up -d
4. **Access the Application**:
   - Backend API and GraphQL Playground: `http://localhost:8080`
   - Frontend Application: `http://localhost:3000`
